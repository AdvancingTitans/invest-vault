# Web reading

For a known public URL, use Agent Reach's active web backend. The zero-configuration
route is Jina Reader:

```bash
curl -s "https://r.jina.ai/URL"
```

Preserve the original URL, page title, publisher, publication date and retrieval time.
Do not treat a blocked page, search page, redirect notice or partial snippet as source
text. Respect the source's access controls and copyright; extract only the minimum facts
needed for the research question.
